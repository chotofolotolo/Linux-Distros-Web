¿Que es Linux?

Linux® es un sistema operativo (SO) open source que creó Linus Torvalds en 1991. Gracias a su comunidad internacional de aficionados, hoy podemos encontrarlo en todo tipo de dispositivos, desde sistemas de puntos de venta hasta las supercomputadoras más potentes del mundo.

Más información sobre Linux

Linux no es un sistema operativo completo, sino un núcleo (kernel) sobre el cual se construyen distintas distribuciones. Es libre, estable y ampliamente utilizado en servidores, computadoras personales y sistemas embebidos.

Código Abierto

El código fuente es libre, permitiendo a cualquiera estudiarlo, modificarlo o distribuirlo.

Distribuciones

Existen múltiples variantes como Ubuntu, Debian, Fedora, Arch y muchas más.

Seguridad

La arquitectura de permisos y comunidad activa lo hacen muy seguro por diseño.







Comandos Básicos de Linux

ls - Lista los archivos y carpetas en el directorio actual.
ls
cd - Cambia de directorio.
cd /home/usuario
pwd - Muestra la ruta del directorio actual.
pwd
mkdir Crea un nuevo directorio.
mkdir nueva_carpeta
rm Elimina archivos o directorios.
rm archivo.txt
sudo Ejecuta un comando como superusuario.
sudo apt update
man Muestra el manual de un comando.
man ls

Estos comandos se escriben en la terminal. Linux es muy potente y flexible, lo que lo hace ideal para muchos tipos de usuarios.

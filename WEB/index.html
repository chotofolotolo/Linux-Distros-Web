¿Que es Linux?

Linux® es un sistema operativo (SO) open source que creó Linus Torvalds en 1991. Gracias a su comunidad internacional de aficionados, hoy podemos encontrarlo en todo tipo de dispositivos, desde sistemas de puntos de venta hasta las supercomputadoras más potentes del mundo.

INFORMACION SOBRE LINUX

Linux es el nombre del kernel (núcleo) utilizado por muchos sistemas operativos modernos. Es el componente central que se encarga de gestionar el hardware de la computadora y proporcionar servicios esenciales a los demás programas. En el uso cotidiano, el término "Linux" suele referirse a todo el sistema operativo, que incluye no solo el kernel, sino también herramientas del proyecto **GNU**, entornos gráficos, servicios y aplicaciones. Este conjunto completo es lo que se conoce como una **distribución Linux** (como Ubuntu, Debian o Arch). Algunos prefieren el término **GNU/Linux** para destacar esta combinación de tecnologías. Linux se caracteriza por ser **libre, seguro, estable, personalizable y eficiente**, lo que lo convierte en una excelente opción para servidores, computadoras personales, dispositivos móviles y sistemas embebidos.







Comandos Básicos de Linux

ls - Lista los archivos y carpetas en el directorio actual.
ls
cd - Cambia de directorio.
cd /home/usuario
pwd - Muestra la ruta del directorio actual.
pwd
mkdir Crea un nuevo directorio.
mkdir nueva_carpeta
rm Elimina archivos o directorios.
rm archivo.txt
sudo Ejecuta un comando como superusuario.
sudo apt update
man Muestra el manual de un comando.
man ls

Estos comandos se escriben en la terminal. Linux es muy potente y flexible, lo que lo hace ideal para muchos tipos de usuarios.

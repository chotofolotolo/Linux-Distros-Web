Distribuciones de Linux y Comandos Básicos

¿Qué es Linux?

Linux es un sistema operativo libre y de código abierto. Se utiliza ampliamente en servidores, computadoras personales y dispositivos embebidos.

Principales Distribuciones de Linux

  • Ubuntu: Ideal para principiantes, basada en Debian.
  • Debian: Muy estable, usada como base para otras distros.
  • Fedora: Patrocinada por Red Hat, incluye software más reciente.
  • Arch Linux: Para usuarios avanzados que quieren configurar todo desde cero.
  • Linux Mint: Basada en Ubuntu, muy amigable para el usuario.
  • Kali Linux: Orientada a pruebas de seguridad y hacking ético.

Comandos Básicos de Linux

ls - Lista los archivos y carpetas en el directorio actual.
ls
cd - Cambia de directorio.
cd /home/usuario
pwd - Muestra la ruta del directorio actual.
pwd
mkdir Crea un nuevo directorio.
mkdir nueva_carpeta
rm Elimina archivos o directorios.
rm archivo.txt
sudo Ejecuta un comando como superusuario.
sudo apt update
man Muestra el manual de un comando.
man ls

Estos comandos se escriben en la terminal. Linux es muy potente y flexible, lo que lo hace ideal para muchos tipos de usuarios.
